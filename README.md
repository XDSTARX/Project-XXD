# Super Surveys in the World - Sistema de Ganhar Dinheiro

## 🚀 Sobre o Projeto

O **Super Surveys in the World** é uma plataforma completa para ganhar dinheiro real através de missões fáceis e rápidas. Os usuários podem ganhar de R$ 0,10 até R$ 20,00 por missão e sacar via PIX para diversos bancos.

## ✨ Funcionalidades Principais

### 💰 Sistema de Ganhos
- **Valores altos**: R$ 0,10 até R$ 20,00 por missão
- **Saque gratuito**: Sem taxas de saque
- **Sem limite**: Usuários podem sacar quando e quanto quiserem
- **Pontos baixos**: Fácil conversão de pontos para dinheiro real

### 🏦 Saque PIX
- **Múltiplos bancos**: PicPay, Nubank, Recargapay, PagBank, InfinitePay, Bradesco, Itaú, Caixa, Santander
- **Processamento rápido**: Saques processados em até 5 minutos
- **Acompanhamento em tempo real**: Sistema de status do processamento
- **Histórico completo**: Todos os saques ficam registrados

### 🎯 Sistema de Missões
- **Missões fáceis**: 2-7 minutos para completar
- **Variadas**: Pesquisas, avaliações, testes de produtos
- **Recompensas imediatas**: Dinheiro creditado instantaneamente
- **Histórico de missões**: Acompanhe todas as missões concluídas

### 🏆 Sistema de Conquistas
- **Conquistas desbloqueáveis**: Primeiro saque, missões completadas, check-ins
- **Recompensas extras**: Bônus em dinheiro por conquistas
- **Progresso visível**: Acompanhe seu progresso

## 🚀 Como Usar

### 1. Acesso
- Abra o arquivo `index.html` no seu navegador
- O sistema funciona offline após carregado

### 2. Cadastro/Login
- Clique em "Entrar" no canto superior direito
- Crie uma conta ou faça login
- Seus dados são salvos localmente

### 3. Ganhar Dinheiro
- Navegue até a seção "Missões"
- Escolha uma missão e clique em "Iniciar Missão"
- Complete a missão (simulada) e ganhe dinheiro instantaneamente

### 4. Sacar Dinheiro
- Vá para a seção "Saque"
- Digite o valor desejado (R$ 0,10 - R$ 20,00)
- Selecione seu banco
- Digite sua chave PIX e nome completo
- Clique em "Solicitar Saque"
- Acompanhe o processamento em tempo real

### 5. Acompanhar Progresso
- Veja seu saldo e pontos no cabeçalho
- Acesse "Histórico" para ver saques e missões
- Verifique "Conquistas" para ver progresso
- Use "Check-in Diário" para ganhar bônus
