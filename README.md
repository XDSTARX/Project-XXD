## 🚀 Instalação e Execução

- NPM ou Yarn

### 1. Instalar Dependências
```bash
npm install
```

### 2. Executar o Servidor
```bash
# Modo desenvolvimento (com auto-reload)
npm run dev

# Modo produção
npm start
```

### 3. Acessar a Aplicação
Abra seu navegador e acesse: `http://localhost:3000`

## 📱 Como Usar

### 1. Cadastro
- Acesse o site
- Clique em "Cadastrar"
- Preencha seus dados (nome, email, senha, telefone para PIX)
- **Ganhe 1000 pontos de bônus** ao se cadastrar!

### 2. Completar Missões
- Vá para a seção "Missões"
- Escolha uma missão disponível
- Clique em "Iniciar Missão"
- Complete a tarefa (assistir vídeo, responder pesquisa, etc.)
- **Ganhe pontos e dinheiro instantaneamente**

### 3. Fazer Check-in Diário
- Acesse diariamente para fazer check-in
- Ganhe pontos progressivos (50, 75, 100, 150, 200, 500, 1000)
- Mantenha sua sequência para bônus maiores

- Vá para "Sacar"
- Escolha o valor (R$ 0,10 a R$ 20,00)
- Selecione seu banco/carteira
- Insira sua chave PIX
- Confirme o saque
- **Receba em 2-5 minutos!**
